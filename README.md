# JS Exam

**JS Exam** — учебный проект по JavaScript, созданный в рамках экзаменационной или практической работы по фронтенд-разработке.

Цель проекта заключалась в закреплении навыков работы с JavaScript, HTML и CSS, а также в создании интерактивного веб-приложения.

## Основные возможности

- динамическое изменение содержимого страницы;
- обработка пользовательских действий;
- работа с DOM;
- валидация данных;
- использование событий;
- управление интерфейсом с помощью JavaScript;
- взаимодействие между HTML, CSS и JavaScript.

## Использованные технологии

- HTML5;
- CSS3;
- JavaScript;
- DOM API;
- Flexbox;
- адаптивная вёрстка.

## Цели проекта

Во время выполнения проекта были отработаны следующие навыки:

- создание структуры страницы;
- работа с элементами DOM;
- обработка событий;
- изменение содержимого страницы;
- работа с формами;
- организация клиентской логики;
- разделение HTML, CSS и JavaScript-кода.

## Запуск проекта

Клонируйте репозиторий:

```bash
git clone https://github.com/DmitriyYakubovskiy/JS_Exam.git
```

Перейдите в папку проекта:

```bash
cd JS_Exam
```

Откройте файл:

```text
index.html
```

Проект не требует установки дополнительных зависимостей и запуска сервера.
